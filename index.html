    Conclusion
    Main takeaways
  • The distance function plays an important role
  • Equiped with a modern text encoder (i.e. BERT), Prototypical Networks reclaim the state-of-the-art spot
  • More recent and complex approaches like Relation or Induction networks perform poorly
  • Traditional classifier baselines are surprisingly strong
    • Future lines of research
  • Are those findings robust to new transformer-based architectures?
  • Can we extend the euclidean distance to a learnable —yet controlled— distance? (e.g. adding weights on dimensions)
  • Can we do better than averageing representations to derive prototypes? (e.g. weighted combination obtained using attention)

    Fold/Unfold method
    Weighting mechanism
    Unfortunately, some pseudo-labels might be wrong. Given a similarity matrix $S$, an unlabeled data point $x_u \in U$ with pseudo-label $\hat{y}$, for each class $c$, we compute its logits $z_{u, c}$ as follows:
    $z_{u, c} = \frac{1}{n_c}\sum\limits_{i=1}^{n_c}\mathbb{1}(y_{i}=c)S_{i, u}$
    We then turn theses scores into probability using a softmax function with a temperature parameter $\tau$
    $\alpha = \frac{\exp(\tau z_{u, \hat{y}})}{\sum_j exp(\tau z_{u, j})}$
    This $\alpha$ parameter will be used in the training loss to penalize pseudo-labels
    On a aggrégé les différentes méthodes de pseudo-labeling, en gardant seulement les pseudo-labels pour lesquelles toutes les méthodes sont d'accord
    Puis, on ampute chacune des méthodes à cette aggrégation. La version amputée qui performe le moins bien, soulignée, montre quelle méthode contribuait le plus à l'aggrégation

    Unsupervised loss
    At each episode, we sample $U$ unlabeled sentences. For each unlabeled sentence $x_{u}$, we generate $M$ paraphrases $\left\{\tilde{x}_{u}^{m}\right\}_{m=1}^{M}$
    Then, we compute the unlabeled prototype $p_{x_{u}}$ as the average embedding of the paraphrases
    $p_{x_{u}} = \frac{1}{M}\sum_{m=1}^{M} f_{\phi}(\tilde{x}^{m}_{u})$
    After obtaining all unlabeled prototypes, we can compute euclidean distances between unlabeled samples and unlabeled prototypes
    $d_{u, v} = \lVert f_{\phi}\left(x_{u}\right) - p_{x_{v}}\rVert_{2}$
    Using softmax, we can turn these into a probability vector!
    $\mathbb{P}_{\phi}\left(u \rightarrow v\right) \propto \text{exp}\left(-d_{u, v}\right)$
    Using these probabilities, we compute an unsupervised loss
    $\tilde{L} = \sum_{u=1}^{U} - \log \mathbb{P}_{\phi}\left(u \rightarrow u\right)$
    Combining supervised and unsupervised losses
    We jointly minimize the supervised loss from prototypical networks $\bar{L}$ and the unsupervised loss $\tilde{L}$
    $L = t^{\alpha} \times \tilde{L} + \left(1 - t^{\alpha}\right) \times \bar{L} \quad ; \quad t \in \left(0, 1\right)$
    The parameter $\alpha$ controls the nature of the shift from supervision to unsupervision

    Conclusion
    Main takeaways
  • We introduced a novel end-to-end intent classification method built around prototypical networks
  • By mimicing the mechanism of prototypical networks, it learns from unlabeled data
  • Using paraphrases which are diverse greatly improve the model's performances
    • Future lines of research
  • Can this extension also benefit other few-shot learning methods?
  • How to further extend the method to the multilingual case?
  • Can we generate more relevant paraphrases using non auto-regressive generative models?
